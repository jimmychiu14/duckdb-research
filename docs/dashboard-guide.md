# DuckDB 股票分析儀表板 — 使用手冊與指標解讀指南

> **網址**：https://jimmychiu14.github.io/duckdb-research/
> **原始碼**：`~/.hermes/workspace/duckdb-research/`
> **資料來源**：TWSE 交易所 API、TPEx 上櫃 API、Yahoo Finance（自動 fallback）
> **更新頻率**：每日自動更新（cron job 驅動）

## 一、基本操作

### 切換股票

在頁面頂部的「監控清單」區塊，點選任意股票連結即可切換。也可以在 URL 加上參數直接指定：

```
https://jimmychiu14.github.io/duckdb-research/?stock=2330
https://jimmychiu14.github.io/duckdb-research/?stock=4979
```

### 新增/移除追蹤標的

編輯專案中的 `stock_watchlist.md` 檔案，在 `## 🎯 當前追蹤標的` 下方增減條目：

```markdown
## 🎯 當前追蹤標的

- 2330 台積電 @twse
- 4979 華星光 @tpex
- 2454 聯發科          ← 新增這行就會自動納入
```

- `@twse` = 上市股票
- `@tpex` = 上櫃股票
- 不標記 = 自動偵測（`@auto`）

存檔後，下次更新腳本跑完就會自動出現在儀表板上。

## 三、指標意義與評估方法

### A. 頂部摘要卡片

| 指標 | 意義 | 評估方法 |
|---|---|---|
| **最新收盤價** | 當日收盤價 | 搭配期間報酬率判斷短線動能 |
| **期間報酬率** | 近 5 日（一週）漲跌幅 | `+` 紅色代表上漲，`-` 綠色代表下跌（台股慣例） |
| **本週區間價** | 近 5 日最高/最低 | 區間越大代表波動越劇烈 |
| **當前距高點回撤** | 目前價格距歷史高點的跌幅 | `-5%` 以內屬高檔震盪；`-20%` 以上屬深度修正 |
| **歷史最大回撤 (MDD)** | 資料期間內最大跌幅 | MDD < -15% 表示該股歷史波動較大 |
| **本週交易總量** | 近 5 日總成交股數/金額 | 量能擴增配合價漲 = 多方力道強 |
| **AI 技術指標燈號** | 均線多空排列判定 | 🔴 看好（多頭排列）/ 🟢 看差（空頭排列）/ 🟡 觀望（糾結） |

### B. 主圖 — K 線與均線

| 指標 | 意義 | 評估方法 |
|---|---|---|
| **收盤價** | 每日收盤折線 | 白色粗線 |
| **5MA** | 5 日移動平均 | 橙色虛線，代表短線趨勢 |
| **20MA** | 20 日移動平均 | 藍色實線，代表中線趨勢 |
| **60MA** | 60 日移動平均 | 綠色實線，代表長線趨勢 |
| **布林上軌** | MA20 + 2×標準差 | 紅色虛線，觸及代表短線過熱 |
| **布林下軌** | MA20 − 2×標準差 | 綠色虛線，觸及代表短線超跌 |

**均線排列判斷法則**：
- **多頭排列**：收盤價 > 5MA > 20MA > 60MA → 趨勢向上
- **空頭排列**：收盤價 < 5MA < 20MA < 60MA → 趨勢向下
- **糾結震盪**：均線交纏 → 方向不明，觀望

**布林通道法則**：
- 價格觸及上軌 → 短線可能過熱，有拉回風險
- 價格觸及下軌 → 短線可能超跌，有反彈機會
- 通道收窄（擠壓） → 即將變盤，注意突破方向

### C. 技術指標面板

#### RSI（相對強弱指標，14 日）

| 數值範圍 | 意義 | 行動參考 |
|---|---|---|
| **> 70** | 超買 🔴 | 短線過熱，注意獲利了結或停利 |
| **50–70** | 偏多中性 | 多方主導但未過熱 |
| **30–50** | 偏空中性 | 空方主導但未超賣 |
| **< 30** | 超賣 🟢 | 短線超跌，可能出現反彈 |

> RSI 適合搭配趨勢判斷使用。在多頭趨勢中 RSI 可能長期維持 50 以上，不宜單獨以 >70 就做空。

#### MACD（指數平滑異同移動平均線）

由三個數值組成：

| 組成 | 計算方式 | 意義 |
|---|---|---|
| **DIF** | EMA(12) − EMA(26) | 快慢線差值，反映短中線動能 |
| **MACD線** | EMA(9) of DIF | DIF 的 9 日平滑，訊號線 |
| **柱狀圖 (OSC)** | DIF − MACD線 | 多空力道的即時變化 |

**判斷法則**：
- **金叉**：DIF 由下往上穿越 MACD 線 → 買進訊號
- **死叉**：DIF 由上往下穿越 MACD 線 → 賣出訊號
- **柱狀圖轉正**（紅色）：多空力道轉強
- **柱狀圖轉負**（綠色）：多空力道轉弱
- DIF 和 MACD 均在零軸上方 → 中期趨勢偏多

#### KD（隨機指標，9 日）

| 組成 | 計算方式 | 意義 |
|---|---|---|
| **K 值** | 2/3 × 前日 K + 1/3 × RSV | 快線，反映當前價格在近期區間的位置 |
| **D 值** | 2/3 × 前日 D + 1/3 × 今日 K | 慢線，K 值的平滑版 |

**判斷法則**：
- **金叉**（↑）：K 由下往上穿越 D → 短線買進訊號
- **死叉**（↓）：K 由上往下穿越 D → 短線賣出訊號
- **K > 80**：短線過熱
- **K < 20**：短線超跌
- 高檔鈍化（K 長期 > 80）：強勢多頭，不宜貿然做空

#### ATR（平均真實波動幅度，14 日）

| 意義 | 評估方法 |
|---|---|
| 衡量每日價格波動的絕對幅度 | ATR 越大 = 波動越劇烈 = 風險越高 |
| 可用於設定停損距離 | 停損可設在 1.5~2 倍 ATR 之外 |

> ATR 不判斷方向，只衡量波動程度。適合用來評估部位風險和設定停損。

#### 布林通道（Bollinger Bands，20 日，2 倍標準差）

同主圖的布林上/中/下軌，此面板顯示最新一日的精確數值。

| 位置 | 意義 |
|---|---|
| 價格在上軌附近 | 短線偏強，可能過熱 |
| 價格在中軌附近 | 趨勢中繼，方向未定 |
| 價格在下軌附近 | 短線偏弱，可能超跌 |

### D. RSI 子圖

紫色折線圖顯示過去 60 日的 RSI 走勢。

| 參考線 | 意義 |
|---|---|
| **紅色虛線 (70)** | 超買閾值 |
| **灰色虛線 (50)** | 多空分界 |
| **綠色虛線 (30)** | 超賣閾值 |

觀察重點：
- RSI 在 50 以上盤整 → 多方主導
- RSI 在 50 以下盤整 → 空方主導
- RSI 頂部/底部背離價格 → 趨勢反轉的早期警訊

### E. 風險與報酬分析

#### 期間報酬率

| 指標 | 計算方式 | 評估方法 |
|---|---|---|
| **月報酬** | 近 22 個交易日漲跌幅 | 正值代表月線收紅 |
| **季報酬** | 近 63 個交易日漲跌幅 | 正值代表季線收紅 |

> 月/季報酬比週報酬更有參考意義，能過濾單週波動雜訊。

#### 年化波動率

| 數值範圍 | 風險等級 | 參考 |
|---|---|---|
| **< 20%** | 低波動 | 適合穩健配置 |
| **20%–35%** | 中等波動 | 一般個股正常範圍 |
| **35%–50%** | 高波動 | 需控制部位 |
| **> 50%** | 極高波動 | 高風險，謹慎操作 |

> 計算方式：日報酬率標準差 × √252。這是年化後的預期波動幅度。

#### 52 週區間

| 指標 | 意義 | 評估方法 |
|---|---|---|
| **52 週最高** | 過去一年最高價 | 接近 = 在山頂附近 |
| **52 週最低** | 過去一年最低價 | 接近 = 在山谷附近 |
| **距高點 %** | 目前價格距 52 週高點的跌幅 | -5% 以內 = 高檔 |
| **距低點 %** | 目前價格距 52 週低點的漲幅 | +5% 以內 = 低檔 |

> 這是判斷股票「貴不貴」的最直觀方式。距高點 -3% 跟距低點 +40% 的意義完全不同。

#### VWAP（成交量加權均價）

| 意義 | 評估方法 |
|---|---|
| 全期間按成交量加權的平均買入成本 | 價格 > VWAP = 多數持有者獲利 |
| 反映市場平均持倉成本 | 價格 < VWAP = 多數持有者虧損 |

> 價格在 VWAP 之上運行通常代表多方主導，反之為空方主導。

#### 連漲跌天數

| 指標 | 意義 | 評估方法 |
|---|---|---|
| **最大連漲** | 歷史最大連續上漲天數 | 連漲越多 = 動能越強，但也越接近反轉 |
| **最大連跌** | 歷史最大連續下跌天數 | 連跌越多 = 賣壓越重，但也可能接近底部 |

#### 支撐/壓力位（近 60 日）

| 指標 | 意義 | 評估方法 |
|---|---|---|
| **支撐位** | 近 60 日最低點 | 價格接近時可能出現買盤進場 |
| **壓力位** | 近 60 日最高點 | 價格接近時可能出現賣壓 |

> 支撐壓力不是精確的點，而是一個區間。搭配成交量觀察突破/跌破的有效性。

#### 量價背離警示

| 狀態 | 意義 | 行動參考 |
|---|---|---|
| **無** | 量價同步 | 正常狀態 |
| **價漲量縮** ⚠️ | 價格上漲但量能萎縮 | 可能反彈乏力，上漲缺乏買盤支撐 |
| **價跌量增** ⚠️ | 價格下跌但量能擴大 | 可能恐慌性賣壓，下跌力道較強 |

> 量價背離是趨勢可能反轉的警訊，但不是絕對訊號，需搭配其他指標確認。

### F. 每日明細表

最新 10 日的完整數值，包含：

| 欄位 | 說明 |
|---|---|
| 日期 / 開盤 / 最高 / 最低 / 收盤 | 基本 OHLC |
| 漲跌 | 當日漲跌點數（▲ 紅漲 / ▼ 綠跌） |
| 成交量 / 金額(億) | 量能數據 |
| **RSI** | 當日 RSI 值 |
| **K / D** | 當日 KD 值 |
| **DIF / MACD** | 當日 MACD 快慢線 |

## 四、指標組合使用建議

### 判斷進場時機

以下條件「同時滿足越多」可信度越高：

- [ ] AI 燈號為 🔴 看好（均線多頭排列）
- [ ] RSI 在 50–70 之間（偏多但未超買）
- [ ] MACD 柱狀圖轉正（多空力道轉強）
- [ ] KD 出現金叉
- [ ] 價格站上布林中軌
- [ ] 量價同步（無背離警示）

### 判斷出場/停利

以下條件出現越多，越應考慮出場：

- [ ] RSI > 70（超買）
- [ ] MACD 出現死叉
- [ ] KD 出現死叉且 K > 80（高檔鈍化後轉弱）
- [ ] 價格觸及布林上軌
- [ ] 量價背離警示出現（價漲量縮）
- [ ] 價格跌破 5MA

### 評估風險

- **年化波動率** > 35% → 高波動，降低部位
- **距 52 週高點** < -3% → 在山頂附近，追高風險大
- **歷史最大回撤** < -15% → 該股歷史波動較大
- **ATR** 大 → 日內波動大，停損要設寬一點

## 五、資料來源與限制

| 來源 | 適用 | 限制 |
|---|---|---|
| **TWSE** | 上市股票（@twse） | 每月第一個交易日可能尚未更新當月資料 |
| **TPEx** | 上櫃股票（@tpex） | 有 Cloudflare 防護，可能失敗 |
| **Yahoo Finance** | 自動 fallback | 成交筆數 (Transactions) 為 0 |

> 當官方來源失敗時，系統會自動切換到 Yahoo Finance，確保資料不中斷。
> 資料涵蓋範圍約 3–6 個月歷史日線，部分指標（如 MACD、52 週）在資料不足時會顯示「—」。
